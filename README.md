简介：
  提供类似hibernate的功能
特点：
  1.支持实体的增删改查功能
  2.支持执行sql语句
    1）获得实体的集合(普通查询)
    2）获得对象数组的集合（分组查询）
缺点：
  1.暂时只支持mysql数据库（也可以手动实现org.tcf.sql.generator.SqlGenerator接口，以便支持其它的数据库）
  2.不支持关系（一对多，多对一，一对一，多对多）
  
注意：
  本项目只是为了个人学习，如果需要尽可使用（使用时，请注明本项目地址。），但并不保证代码完全正确可靠。
